# IPTV TV App 📺

Um aplicativo web moderno para assistir canais IPTV diretamente no navegador.

## ✨ Funcionalidades

- Suporte a listas M3U
- Interface estilo TV moderna
- Busca de canais
- Agrupamento por categoria
- Suporte a streams HLS (m3u8)
- Responsivo para mobile
- Salva configurações localmente

## 🚀 Como usar

1. Clone o repositório
2. Configure sua URL M3U nas configurações
3. Selecione um canal e assista

## 🔧 Configuração

Clique no ícone de engrenagem e adicione:
- URL da playlist M3U (obrigatório)
- URL do EPG (opcional)
